[//]: # (## پایگاه داده چیست؟)

[//]: # ()
[//]: # (پایگاه داده یا Database مجموعه‌ای از داده‌ها است که به صورت ساختارمند درون کامپیوتر نگهداری می‌شوند.)

[//]: # (سامانه مدیریت پایگاه داده یا DBMS&#40;Database Management System&#41; نرم‌افزاری است که با کاربر نهایی، برنامه‌های کاربردی و خود پایگاه داده برای ذخیره و بازیابی داده تعامل می‌کند.)

[//]: # ()
[//]: # (DBMS های مختلفی برای کابردهای متفاوت وجود دارند که پرکاربردترین آن‌ها، DBMS های رابطه‌ای یا همان RDBMS &#40;Relational Database Management System&#41;ها هستند. در این DBMSها داده‌ها به صورت سازمانمند در جداول ذخیره می‌شوند و مقادیر موجود در هر جدول با یکدیگر ارتباط دارند. به همین خاطر به آن رابطه‌ای با Relational گفته می‌شود.)

[//]: # ()
[//]: # (از آنجایی که اکثر RDBMSها از زبان پرسمان سازمان‌یافته یا SQL&#40;Structured Query Language&#41; استفاده می‌کنند، با آشنایی با یکی از این RDBMSها می‌توانید با صرف زمان کم نحوه استفاده از یک RDBMS جدید را فرابگیرید. همچنین SQL جزو استاندارد ANSI و ISO می‌باشد. تعداد زیادی RDBMS وجود دارد که بسته به نیاز از آن‌ها استفاده می‌شود.)

[//]: # ()
[//]: # (MySQL, SQL Server, PostgreSQL و SQLite نمونه‌هایی از RDBMSهای معروف هستند. در این فاز با PostgreSQL کار می‌کنید که یک DBMS قدرتمند و open source می‌باشد.)
